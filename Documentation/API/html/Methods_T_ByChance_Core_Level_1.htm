AddChunk
Adds a chunk to this level.

This is done by aligning the given open context of an existing level chunk with the open context of the new chunk and then adding the chunk to the chunk list.

Note that the new chunk is assumed to fit the level geometry. See FitsLevelGeometry(Context, Context) for further information on how to check this first.
